## Pro vs Basic components

The Basic tier ships every component you need to build a working storefront. Pro unlocks components for richer interaction:

| Component | Basic | Pro | Scale |
| --- | :-: | :-: | :-: |
| Product Gallery / Price / Variant Picker | ✓ | ✓ | ✓ |
| Add to Cart / Cart Button / Checkout | ✓ | ✓ | ✓ |
| Cart Item List / Row / Image / Title | ✓ | ✓ | ✓ |

## Activating a license key

After buying a Pro or Scale license you'll receive an email with your activation key.

<img src="/images/pricing-activate-key.png" alt="Frammerce pricing page - Activate Key entry point" />

<Steps>
  <Step title="Open the Pricing page">
    In the plugin panel, click **Pricing**.
  </Step>
  <Step title="Click Activate Key">
    The "Already have a license key?" card shows an **Activate Key** button. Click it.
  </Step>
  <Step title="Paste your key">
    Paste the license key from your purchase email and submit.
  </Step>
  <Step title="Plan unlocked">
    The Pricing page now shows the active tier, and your panel header reflects the new plan.
  </Step>
</Steps>

## Removing a license

You may need to remove a license to move it to a different account.

<Warning>
You can't remove a license while a Shopify store is connected. The cart runtime depends on the plan info to gate features (e.g. checkout). Disconnect the store from **Settings → Disconnect Store** first.
</Warning>

<Steps>
  <Step title="Disconnect Store">
    Settings → **Disconnect Store** (click twice to confirm).
  </Step>
  <Step title="Open Pricing">
    Pricing → your license key card.
  </Step>
  <Step title="Click Remove">
    The red **Remove** button. Confirm.
  </Step>
</Steps>
